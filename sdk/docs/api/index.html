Tribes by Astrix SDK

A comprehensive SDK for integrating with the Tribes by Astrix platform, allowing organizations to implement a token-based points system, community management, and engagement tools.

Features

  • Astrix Token Integration: Easily integrate with Astrix tokens for your organization's points system
  • Point System Management: Award, transfer, and manage points for users within your community
  • Tribe Management: Create and manage tribes, members, and permissions
  • User Profiles: Handle user profiles and authentication
  • Content Management: Support for posts, comments, and other content types with multiple formats
  • Analytics: Track user engagement and community metrics

Architecture Overview

The SDK follows a modular architecture making it easy to use and extend:

  • Main SDK Class: Central entry point that manages connections and initializes all modules
  • Module-based Design: Functionality is organized into specialized modules (content, points, token, tribes, etc.)
  • Type Safety: Comprehensive TypeScript interfaces and enums ensure type safety across the SDK
  • Error Handling: Standardized error types and handling across all modules

Installation

npm install tribes-by-astrix-sdk

Frontend Integration Guide

React Integration

import { useEffect, useState } from 'react';
import { AstrixSDK } from 'tribes-by-astrix-sdk';
import { ethers } from 'ethers';

function App() {
  const [sdk, setSdk] = useState(null);
  const [connected, setConnected] = useState(false);
  const [posts, setPosts] = useState([]);

  // Initialize SDK
  useEffect(() => {
    const initSDK = async () => {
      // For MetaMask or similar wallets
      if (window.ethereum) {
        // Configure contract addresses based on your deployment
        const sdkInstance = new AstrixSDK({
          provider: window.ethereum,
          contracts: {
            roleManager: '0x...',
            tribeController: '0x...',
            astrixToken: '0x...',
            tokenDispenser: '0x...',
            astrixPointSystem: '0x...',
            postMinter: '0x...',
            postFeedManager: '0x...'
          },
          verbose: true
        });
        
        setSdk(sdkInstance);
      }
    };
    
    initSDK();
  }, []);

  // Connect wallet
  const connectWallet = async () => {
    try {
      if (sdk) {
        await window.ethereum.request({ method: 'eth_requestAccounts' });
        const provider = new ethers.BrowserProvider(window.ethereum);
        const signer = await provider.getSigner();
        await sdk.connect(signer);
        setConnected(true);
      }
    } catch (error) {
      console.error('Connection error:', error);
    }
  };

  // Fetch posts from a tribe
  const fetchPosts = async () => {
    if (sdk && connected) {
      try {
        const result = await sdk.content.getPostsByTribe({
          tribeId: 1,
          limit: 10,
          includeDetails: true
        });
        
        setPosts(result.posts || []);
      } catch (error) {
        console.error('Error fetching posts:', error);
      }
    }
  };

  return (
    <div>
      {!connected ? (
        <button onClick={connectWallet}>Connect Wallet</button>
      ) : (
        <>
          <h2>Connected!</h2>
          <button onClick={fetchPosts}>Fetch Posts</button>
          
          <div>
            {posts.map(post => {
              // Parse JSON metadata
              const metadata = JSON.parse(post.metadata);
              
              return (
                <div key={post.id}>
                  <h3>{metadata.title}</h3>
                  <p>{metadata.content}</p>
                </div>
              );
            })}
          </div>
        </>
      )}
    </div>
  );
}

export default App;

Creating a Tribe

// Create a new tribe
const tribeId = await sdk.tribes.createTribe({
  name: "My Amazing Community",
  description: "A community for enthusiasts",
  metadata: JSON.stringify({
    avatar: "https://example.com/avatar.png",
    banner: "https://example.com/banner.png",
    socialLinks: {
      twitter: "https://twitter.com/mycommunity",
      discord: "https://discord.gg/mycommunity"
    }
  }),
  joinType: "PUBLIC", // or "WHITELIST", "PAID", "COLLECTIBLE"
});

console.log(`Created tribe with ID: ${tribeId}`);

Points Module

// Award points to a user for creating a post
await sdk.points.awardPoints({
  tribeId: 1,
  member: "0xUserAddress",
  points: 50,
  actionType: "POST"
});

// Get user's point balance
const balance = await sdk.points.getPoints({
  tribeId: 1,
  member: "0xUserAddress"
});
console.log(`User has ${balance} points`);

// Set point rewards for different actions
await sdk.points.setActionPoints({
  tribeId: 1,
  actionType: "POST",
  points: 50
});

await sdk.points.setActionPoints({
  tribeId: 1,
  actionType: "COMMENT",
  points: 10
});

Error Handling

The SDK provides standardized error handling:

try {
  const result = await sdk.content.createPost({
    tribeId: 1,
    metadata: JSON.stringify({
      title: "Hello World",
      content: "This is my first post!",
      type: "TEXT"
    })
  });
} catch (error) {
  if (error.type === "CONTRACT_ERROR") {
    console.error("Smart contract error:", error.message);
    // Handle contract errors (gas, reverts, etc.)
  } else if (error.type === "VALIDATION_ERROR") {
    console.error("Validation error:", error.message);
    // Handle validation issues
  } else if (error.type === "UNAUTHORIZED") {
    console.error("Authorization error:", error.message);
    // Handle permission issues
  } else {
    console.error("Unknown error:", error);
  }
}
